# Blueway

Blueway is an open source cross-platform backup program.

## Features:
 - [ ] Backup files & folders
 - [ ] Encryption & Compression of each file
 - [ ] Hash verify files
 - [ ] Backup application data.
 - [ ] Install apps back.
 - [ ] Automate backups.
 - [ ] Upload/download backup to cloud.
 - [ ] Backup to drive.
 - [ ] Sync capabilities.
